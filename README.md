## 📌 What is ScaffoldHQ?

ScaffoldHQ is a free, open student platform for:

- 📓 Taking and organising **notes** by subject and stream
- 💼 Building and showcasing **projects** with version history
- 📊 Tracking **skills** across Science, Arts, Commerce, and Law
- 🔗 **Publishing** work to LinkedIn, Google Sheets, and Google Drive
- 🔒 Controlling **public vs private** visibility per project
- 💬 **Messaging** other student creators to collaborate
- 🚫 **Blocking and reporting** harmful profiles
- © **Copyright-protecting** every piece of work automatically

## 🚀 Deploy to GitHub Pages (3 minutes)

### Option A — Upload via GitHub Web (No coding needed)

1. Go to [github.com](https://github.com) → click **"New repository"**
2. Name it `scaffoldhq` → set to **Public** → click **"Create repository"**
3. Click **"uploading an existing file"**
4. Upload **`index.html`**, **`README.md`**, **`sitemap.xml`**, and **`404.html`**
5. Click **"Commit changes"**
6. Go to **Settings → Pages**
7. Source: **Deploy from a branch** → Branch: `main` → Folder: `/ (root)` → **Save**
8. Wait ~60 seconds → your site is live at `https://yourusername.github.io/scaffoldhq`

### Option B — Git CLI

```bash
git clone https://github.com/yourusername/scaffoldhq.git
cd scaffoldhq
# copy all files into this folder
git add .
git commit -m "Launch ScaffoldHQ"
git push origin main
```

Then enable Pages in Settings as above.
